Menyalakan Codage

Saya merasakan dorongan untuk memisahkan postingan yang berhubungan dengan tekonologi dari tumblr ke tempat yang lebih pas. Ada dua alasan, penyorotan untuk sintaks dari kode dan lokalisasi bahasa. Saya tahu bisa menggunakan gist untuk menulis kode di tumblr, tapi saya ingin fleksibilitas yang lebih lagi.

Setelah meriset beberapa opsi, akhirnya saya memutusakn untuk menggunakan Middleman. Alasannya;

  • platformnya aktif, dibuat dan dikelola oleh orang-orang jago di thoughbot
  • berbasis ruby, yang memang ingin saya pelajari lebih lanjut
  • mendukung fitur penyorotan kode dan bahasa lebih dari satu
  • bisa di-deploy dengan github pages

Middleman menyediakan template untuk mempercepat pembuatan blog. Walau dengan template namun website tetap bisa dikostumisasi lebih lanjut. Takarannya ini yang pas untuk saya, seimbang antara kemudahan dan flesibilitas.

Ada pedoman yang asik untuk menata blog middleman Anda. Saya sangat menyarankannya jika ingin melakukan penataan lebih lanjut ke situs middleman Anda, bahkan pedoman ini yang saya gunakan untuk menata website ini menyerupai blog utama saya.

Untuk info lebih lanjut dengan kakas-kakas middleman, bisa cek Bourbon (Sass mixin library), Neat (grid framework) dan Bitters (konfigurasi standar untuk Bourbon).

Kesimpulan

Walaupun banyak yang harus di-setup, tapi saya puas dengan hasilnya. Ga jelek-jelek amat lah untuk rubyist pemula seperti saya, untuk bisa membuat, menata dan menjalankan website saya. Cek juga kode sumber dari blog ini kali-kali tertarik.
